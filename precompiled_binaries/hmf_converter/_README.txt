Overview:
This is a tool to convert model files to the HMF format. It will also convert any referenced textures to DDS.

Usage:
HMFConverter.exe -f <input filePath>

Optional parameters:
    -i <ini filePath>
    -o <output filePath>"


Other Usage:
HMFConverter.exe -c <iniOutput filePath>
    
    to generate an ini file with default values
